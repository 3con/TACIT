Install From Update Site

Installing the Latest Release: 1.0.0

To install via one of the update site URLs listed below, copy and paste it into the “Help > Install new software” dialog.

TACIT can be installed in the following ways:

  • Main Update Site: http://cssl.usc.edu:8081/tacit/updates (Recommended)
  • Nightly Build: http://cssl.usc.edu:8081/tacit/updates
