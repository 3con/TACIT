Basic Tutorial: Collecting data using Latin Crawler
Specifying Output Folder
First, you will need to specify an output path folder where the downloaded .txt files should be stored. A path can be selected by clicking on the "Browse" button to the right of the Output Location box. Note: The program will create a new folder for each author name where it will save all files by that author. Folders will also be generated for document files from different works.
Document Selection
Next you will need to specify the author(s) for which you would like to request documents by clicking on the "Add" button to the right of the Author Details box. Authors can be selected one at a time, or you can select multiple authors by clicking their names while holding down the "ctrl" button. You can select a range of authors by holding down "ctrl" "shift". You can also remove authors by selecting their name in the Author Details box and clicking the "Remove" button.
Understanding Latin Crawler Output


The data output will be in .txt text file format, with a separate .txt for each document. Documents will be organized by Author, with subfolders for different works.
