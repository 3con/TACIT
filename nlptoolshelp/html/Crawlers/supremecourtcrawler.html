United States Supreme Court Crawler
Overview
The US Supreme Court Crawler is a program that automatically scans and collects court case transcription data from the website Website and writes that data into text files that are readable for automated text analysis techniques. This crawler also saves meta data about the speeches including XXX, and can also download the mp3 files.
