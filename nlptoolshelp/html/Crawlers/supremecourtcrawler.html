United States Supreme Court Crawler
Overview
The US Supreme Court Crawler is a program that automatically scans and collects court case transcription data from the IIT Chicago-Kent College of Law Supreme Court Case Website and writes that data into text files that are readable for automated text analysis programs. This crawler also saves meta data about the speeches including XXX, and can also download the audio mp3 files.

Basic Tutorial: Collecting data using the US Supreme Court Crawler
Supreme Court Speech Selection
First, you will need to determine which Supreme Court cases you would like the crawler to collect by selecting a Filter Type.

Selecting the Term Filter Type will allow you to select cases by calendar year. Cases are grouped by single year and also by decade.

Selecting the Issues Filter Type will allow you to select cases by topic or social issue of interest.

Specifying Output Folder
Next, you will need to specify an output path folder where the downloaded .txt files should be stored. A path can be selected by clicking on the "Browse" button to the right of the Output Location box. Note: The program does not create a new folder for the files it creates. If you would like your files saved together into a single folder, we recommend creating a new folder for the output in advance and selecting that folder as the output location.
Audio Files
To download the .mp3 audio recordings of the court cases along with the transcriptions, check the box nex to "Download Audio". As these files tend to be larger, you also have the option to check the "Truncate" box which will truncate all audio files (which does xxxxx).

Understanding Senate Crawler Output
The court case transcription output will be in .txt text file format, with a separate .txt for each document/case. File Names xxxx. Documents will be organized by xxx. The program will also output a summary .csv file with information about all generated .txt files which includes (when available): case name, docket number, date argued, date decided, majority author, vote, file type (transcript or audio), and the file name. The summary file name will be supremecourt-[time stamp].csv.
