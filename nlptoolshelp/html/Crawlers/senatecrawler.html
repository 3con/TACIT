United States Senate Crawler
Overview
The United States Senate Crawler is a program that automatically scans and collects speech transcription data from the Library of Congress THOMAS Website and writes that data into text files that are readable by automated text analysis programs. This crawler also saves meta data about the speeches including the name of the speaker, the date of the speech and xxx.

Basic Tutorial: Collecting data using the Senate Crawler
What each button does and what you need to specify.
Requesting Data and Specifying Output Folder
Selecting Speeches
First, you will need to determine which Senate speeches you would like the crawler to collect under the Input Parameters panel. First, select the Congress you would like to crawl from the drop down menu. This selection will then update the Senator drop down menu to include only the senators in that Congress. Speeches can be collected for individual Senators, all Senators, or for All Senators of a particular party affiliation (i.e., Republican, Democrat, Independent).

Limiting Records
Checking the box next to Limit Records per Senator in the Limit Records panel allows users to specify crawl a limit the number of speech transcriptions returned by the crawler by typing the number of speeches requested in the No.of.Records per Senator box. If Sort Records by Date is set to yes, the crawler will return the number of speeches requested in chronological order starting from the most recent. If no is selected, the crawler will return the number of speeches requested as a random subset of that Senator's records.
Output Path
To specify an output folder where crawled files will be saved, click on the Browse button to the right of the Output Location bar and select a folder. Note: The program does not create a new folder for the files it creates. If you would like your files saved together into a single folder, we recommend creating a new folder for the output in advance and selecting that folder as the output location.

Understanding Senate Crawler Output
The Console panel at the bottom of the tool will provide information about the congress member selected, the website url being crawled, and the names of the files being created while the crawler is running. Once complete, the data gathered will be in .txt text file format, with a separate .txt for each document. The generated text file name format is: Congress#-Senator last name-party affiliation-state affiliation-date-title (e.g., 114-Ayotte-R-NH-June 10 2015-NATIONAL DEFENS-355.txt)

The program will also create a file called senate-summary-[date-time].csv . This .csv file provides a summary of all files created with additional documentation about each file including (when available): Congress number, Speech Date, Senator Name, Political Affiliation, US State Affiliation, Title of Speech, and the downloaded file name.
