United States Senate Crawler
Overview
The United States Senate Crawler is a program that automatically scans and collects speech transcription data from the website Website and writes that data into text files that are readable for automated text analysis programs. This crawler also saves meta data about the speeches including the name of the speaker, the date of the speech and xxx.

Understanding Senate Crawler Output
The data output will be in .txt text file format, with a separate .txt for each document. Documents will be organized by Senate session in separate folders. The program will also output a .csv file with a list of all files created.
