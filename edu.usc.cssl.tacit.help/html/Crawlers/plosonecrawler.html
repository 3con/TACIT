TACIT User Guide

TACIT PLOS Crawler
Overview
The PLOS Crawler tool collects text from the PLOS ONE journal website. PLOS is a website that provides an open access platform for scientific research articles to be published in an impactful and accessible setting. The crawler provides various filtering settings including a word filter option and an author filter option. Users can choose the specific attributes they want collected with their data, and can also set a maximum limit for the number of documents they want to be crawled. The data is saved in a format that is accessible and analyzable by the TACIT tool as well as other language analysis algorithms.

Basic Tutorial: Collecting data using PLOS Crawler
PLOS Crawler Key
In order to crawl data using the PLOS crawler, you must first create an account on the PLOS website and get an api key. The first time you click the green run arrow, TACIT will prompt you to enter your api key. Instructions for creating an account and receiving the api key can be found here: http://api.plos.org/registration/.
PLOS Crawler Filters
Typepad data can be crawled based on keywords found in the blog content (Content Word Search) or the blog title (Title Word Search). The Word Filter option allows you to crawl for journal article content using a specific keyword or set of keywords. For multiple keywords, separate each word by entering a semicolon in between each word.

The Author Filter allows you to crawl for journal articles written by a specific author.

PLOS Crawler Limits
Max Limit: You can limit the number of articles collected using the Max Limit box. The crawler will default to collecting 100 journal articles if you select the limit number of documents option and do not specify a number.

Specifying Journal Article Attributes for Collection
Users can select which types of information they would like the crawler to collect about each journal article by checking the box next to the following attributes:
Title: Article title
Abstract: the article abstract
Publication Date: the date of article publication
Author: The names of the article's authors
Subject: subject domains assigned to the article by PLOS
Journal: the PLOS journal the article was published in
Body: The entire article text
Introduction: the text from the introduction section of the article
Materials and Methods: The text from the materials and methods sections of the article
Results and Discussion: the text from the results and discussion sections of the article
Conclusion: the text from the conclusion section of the article
Score: the score assigned to the article indicating the quality of the paper based on various qualifications such as how many times it has been referenced in other publications.
Specifying Output Details
By default, all collected text in TACIT is saved as a corpus in Corpus Management. Specify the name you would like to assign this corpus in the Corpus Name box under Output Details .
Understanding PLOS ONE Crawler Output
The Console panel at the bottom of the tool will provide information about the Supreme Court term/issue selected, the website url being crawled, and the names of the files being created while the crawler is running. Crawl data is saved in Corpus Management under the corpus name you provided. Data can be output into formats compatible with other programs using Corpus Management.
