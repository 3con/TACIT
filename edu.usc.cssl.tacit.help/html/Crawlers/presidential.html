TACIT User Guide

US Santa Barbara Presidential Papers Crawler
Overview
The UC Santa Barbara Presidential Papers Crawler collects text from The American Presidency Project Website and writes that data into text files that are readable by automated text analysis programs.

Basic Tutorial: Collecting data using the US Santa Barbara Presidential Papers Crawler
Selecting Crawling Details
The presidential crawler can collect data using either a keyword search, or when no keyword is selected, using other filtering features such as the President or time frame.

Keyword Search: Presidential papers can be crawled based on a single keyword or a combination of two keywords. To search and collect papers including one keyword, supply the keyword in the keyword 1 box. If you are interested in two keywords, you can provide a second keyword in the Keyword 2 box. If using two keywords, you can select (and) to only collect documents with both terms present, (or)to select documents with at least one of the keywords present, or (not)to select documents that include the first keyword but not the second keyword.

Additional Crawler Filters: Regardless of whether or not a keyword is supplied, you may also select from the following additional filters and restrictions to tailor the results crawled from the presidential papers.
Specify Date Range and Specify Date: By default, neither a date range or a date is specified when neither of the boxes are checked, regardless of the “From:” and “To:” fields being filled. Checking the Specify Date Range box allows you to enter in a start and end date for desired articles. You may either type in the date in the format MM/DD/YYYY, or use the drop down button to select start and end dates. Checking the Specify Date box allows you to enter one specific date for crawling.
Select President: The Select President field allows you to select the president that you would like to crawl for. You may also choose to select the “President” option (the default), which will crawl all papers from all presidents.
Select Document Category: The Select Document Category allows you to specify the type of document you would like to crawl for. This includes various types of oral documentation, writings, debates, and more.
Selecting Crawling Limits
You may opt to check the box to Limit the Results. Selecting this option then allows you to type in the maximum number of papers that you would like to download.
Specifying Output Details
By default, all collected text in TACIT is saved as a corpus in Corpus Management. Specify the name you would like to assign this corpus in the Corpus Name box under Output Details .
Understanding Presidential Paper Crawler Output
The Console panel at the bottom of the tool will provide information about the presidential paper selected, the website url being crawled, and the names of the files being created while the crawler is running. Crawl data is saved in Corpus Management under the corpus name you provided. Data can be output into formats compatible with other programs using Corpus Management.
