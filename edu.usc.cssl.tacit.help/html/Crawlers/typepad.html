TACIT User Guide

TACIT Typepad Blog Crawler
Overview
The Typepad Blog Crawler tool collects text from the Typepad website. Typepad is a website that publishes blogs, and this crawler allows users to crawl blog posts with various filtering options including keyword search for titles, blog post text, or both. The crawler will collect data from the most recent relevant blogs, the most relevant blogs, the most recent blogs, or the least recent blogs. The data is then saved in a format that is accessible and analyzable by the TACIT tool as well as other language analysis programs.

Basic Tutorial: Collecting data using Typepad Crawler
Typepad Crawler Filters
Typepad data can be crawled based on keywords found in the blog content (Content Word Search) or the blog title (Title Word Search). The Content Word Search allows you to crawl for blog post content using a specific keyword or set of keywords. For multiple keywords, separate each word by entering a semicolon in between each word. This option will search for your keyword(s) anywhere in the text of the blog post.

The Content and Title Word Search allows you to crawl for blog post content and titles using a specific keyword or set of keywords. For multiple keywords, separate each word by entering a semicolon in between each word. This option will search for your keyword(s) anywhere in the text of the blog post and in blog post titles.
Crawler Limits
The Crawl Order option allows you to specify the time frame you want to collect blog posts from.
      The Most Recent Relevant option will collect the blog posts that were published most recently and had the highest number of views.
      The Most Relevant option will collect the blog posts that had the highest number of views.
      The Most Recent option will collect the blog posts that were published most recently.
      The Least Recent option will collect the blog posts that were published least recently.

Max Limit: You can also limit the number of blog posts collected. You can enter the maximum number of posts you want collected in the box next to the Max Limit option. The crawler will use the number 10 by default if you select this option and do not specify a number.

Specifying Output Details
By default, all collected text in TACIT is saved as a corpus in Corpus Management. Specify the name you would like to assign this corpus in the Corpus Name box under Output Details .
Understanding Typepad Crawler Output
The Console panel at the bottom of the tool will provide information about the Supreme Court term/issue selected, the website url being crawled, and the names of the files being created while the crawler is running. Crawl data is saved in Corpus Management under the corpus name you provided. Data can be output into formats compatible with other programs using Corpus Management.
