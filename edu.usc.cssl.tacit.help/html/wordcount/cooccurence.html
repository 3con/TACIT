Select output files save folder. After specifying all parameters, click the green and white play button located in the top right corner of the window to run the program. Output information will display in the console panel at the bottom of the tool.
Understanding Co-Occurrence Output
The data output will be in .csv file format. The file name includes the type of technique used for analysis and the time stamp for when the analysis was completed.
