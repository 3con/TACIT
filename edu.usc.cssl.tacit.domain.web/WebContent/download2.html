Get TACIT

Download the latest version of TACIT, a free and Open Source Text Analysis software.

Citation - Tacit: An Open-Source Text Analysis, Crawling and Interpretation Tool(September 15, 2015)
Dehghani, Morteza and Johnson, Kate M. and Garten, Justin and Balasubramanian, Vijayan and Singh, Anurag and Shankar, Yuvarani and Rajkumar, Aswin and Parmar, Niki Jitendra and Hoover, Joe and Pulickal, Linda and Boghrati, Reihane
Available at SSRN: http://ssrn.com/abstract=2660651

Required configuration

  • 2 GB of main memory
  • At least 512 MB of free space on the hard drive for extraction
  • JRE version 6 (or later)

Installation Steps

  • Install JRE as mentioned in JRE Installation Guide.
  • Make sure to install 32-bit JRE for 32-bit TACIT and 64-bit JRE for 64-bit TACIT.
  • Download the TACIT zip file and unzip it.
  • To run TACIT:
    • For Windows & Linux: Double click the TACIT.exe file in the unzipped folder.
    • For MAC: Press CTRL key and click to open the TACIT file.
