				<h2>
					<i class="fa fa-check-circle"></i> Installation Steps
				</h2>
				<ul>
					<li>Install JRE as mentioned in <a
						href="http://www.oracle.com/technetwork/java/javase/downloads/index.html">JRE
							Installation Guide.</a></li>
					<li>Make sure to install 32-bit JRE for 32-bit TACIT and
						64-bit JRE for 64-bit TACIT.</li>
					<li>Download the TACIT zip file and unzip it.</li>
					<li>To run TACIT:
						<ul>
							<li>For Windows & Linux: Double click the TACIT.exe file in
								the unzipped folder.</li>
							<li>For MAC: Press CTRL key and click to open the TACIT
								file.</li>
						</ul>
					</li>
